API

Typeahead

成员 值类型 默认值 描述
value String ''
data Array 本地数据
async String 异步http url请求
limit Number 8 最多显示数量
key String null
match-case Boolean false
match-start Boolean false 从开始匹配，如果是true,输入'a',则匹配'ab'而不会匹配'ba'
on-hit Function click或者hit的回调函数
template String <span v-html="$value | highlight query"></span> 用于渲染输入请求
