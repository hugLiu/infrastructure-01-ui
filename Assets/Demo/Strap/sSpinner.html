Spinner

用于页面和区块的loading状态。

何时使用

  • 页面局部处于等待数据渲染时。

组件演示

基本


                    
        <so:s_spinner  v-ref:spinner size="md" text="I will close in 2 secs"></so:s_spinner>
        <button class="btn btn-ifno" @click="show">show</button>

API

成员 值类型 默认值 描述
size String(sm,md,lg,xl) md 动画尺寸
fixed Boolean false 设置动画效果是否覆盖全屏
