Tabs

导航标签页。


组件演示

基本用法


空山不见人，但闻人语响。


返影入深林，复照青苔上。

红豆生南国，春来发几枝。


愿君多采撷，此物最相思。

 
                    
         <so:s_tabset :active="1">
            <so:s_tab header="鹿柴">
               <h4>空山不见人，但闻人语响。</h4> <br>
                <h4>返影入深林，复照青苔上。</h4>
            </so:s_tab>
            <so:s_tab header="相思">
                <h4>红豆生南国，春来发几枝。</h4><br>
                <h4>愿君多采撷，此物最相思。</h4>
            </so:s_tab>
        </so:s_tabset>

参数列表

so:s_tabset

参数 值类型 默认值 描述
active Number 0 展示序列号为active的tab内容

so:s_tab

参数 值类型 默认值 描述
disabled Boolean false 是否能够通过click事件展示tab内容
header String tab的标题
