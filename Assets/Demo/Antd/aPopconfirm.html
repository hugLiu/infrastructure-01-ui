Popconfirm

点击元素，弹出气泡式的确认框。

何时使用

  • 目标元素的操作需要用户进一步的确认时，在目标元素附近弹出浮层提示，询问用户。
  • 和 confirm 弹出的全屏居中模态对话框相比，交互形式更轻量。

组件演示

进度条


删除 
                    
<so:a_popconfirm :title="text"
                 :on-confirm="_confirm"
                 :on-cancel="_cancel">
    <a href="javascript:;">删除</a>
</so:a_popconfirm>

动态展示


上左 上边 上右 
                    
<so:a_popconfirm :placement="'topLeft'" :title="text" :on-confirm="_confirm">
    <so:a_button>上左</so:a_button>
</so:a_popconfirm>
<so:a_popconfirm :placement="'top'" :title="text" :on-confirm="_confirm">
    <so:a_button>上边</so:a_button>
</so:a_popconfirm>
<so:a_popconfirm :placement="'topRight'" :title="text" :on-confirm="_confirm">
    <so:a_button>上右</so:a_button>
</so:a_popconfirm>

API

参数 说明 类型 默认值
placement 气泡框位置，可选 top/left/right/bottom string top
title 确认框的描述 string 无
onConfirm 点击确认的回调 function 无
onCancel 卡片内容 function 无
