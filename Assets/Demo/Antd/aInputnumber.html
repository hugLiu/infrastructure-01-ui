InputNumber

通过鼠标或键盘，输入范围内的数值。

何时使用


当需要获取标准数值时。

组件演示

基本用法


                    
<so:a_checkbox :default-checked="false" :on-change="_change"><span>Checkbox</span></so:a_checkbox>

基本用法


                    
<so:a_checkbox :default-checked="false" :on-change="_change"><span>Checkbox</span></so:a_checkbox>

基本用法


Toggle disabled 
                    
<so:a_checkbox :default-checked="false" :on-change="_change"><span>Checkbox</span></so:a_checkbox>

不可用


                    
<so:a_checkbox :default-checked="false" :disabled="true"><span>不可用</span></so:a_checkbox>
<so:a_checkbox :default-checked="true" :disabled="true"><span>不可用</span></so:a_checkbox>

API

参数 说明 类型 默认值
min 最小值 Number Infinity
max 最大值 Number Infinity
value 当前值 Number
step 每次改变步数 Number 1
defaultValue 初始值 Number
onChange 变化回调 Function
disabled 禁用 Boolean false
size 输入框大小 String 无
