Tag

进行标记和分类的小标签。

何时使用

  • 用于标记事物的属性和维度。
  • 进行分类。

组件演示

基本


标签一 标签二 标签三 标签四链接 
                    
<so:a_tag>标签一</so:a_tag>
<so:a_tag>标签二</so:a_tag>
<so:a_tag closable :on-close="_onClose">标签三</so:a_tag>
<so:a_tag :href="'http://www.baidu.com'">标签四链接</so:a_tag>

四种颜色类型


蓝色 绿色 黄色 红色 
                    
<so:a_tag closable :color="'blue'">蓝色</so:a_tag>
<so:a_tag closable :color="'green'">绿色</so:a_tag>
<so:a_tag closable :color="'yellow'">黄色</so:a_tag>
<so:a_tag closable :color="'red'">红色</so:a_tag>

API

参数 说明 类型 可选值 默认值
href 链接的地址，会传给 a 标签 string false
closable 标签是否可以关闭 boolean false
on-close 组合时根据此项判定checked function 无
color 标签的色彩 string blue green yellow red 无
